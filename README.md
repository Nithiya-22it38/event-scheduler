# Event Scheduling & Resource Allocation System

Flask application for scheduling events and allocating shared resources with conflict detection.

# feature
- Add/Edit/View Events
- Add/Edit/View Resources (rooms, instructors, equipment)
- Resource allocation with time conflict detection
- Resource utilization reports
